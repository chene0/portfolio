<task type="auto">
  <name>Task 2: Make experience cards expandable with Framer Motion</name>
  <files>src/components/experiences/experience.tsx</files>
  <action>
    Rewrite the Experience component to be an expandable card using `useState` + Framer Motion.

    **Imports to add:**
    ```ts
    import { useState } from 'react'
    import { motion, AnimatePresence } from 'framer-motion'
    import { ChevronDown } from 'lucide-react'
    ```
    Check if lucide-react is available; if not, use a simple SVG chevron inline.

    **Component structure:**
    ```
    <div className="flex flex-col" onClick toggle>
      {/* Always visible header */}
      <div className="flex items-start gap-4 cursor-pointer group">
        <logo image />
        <div className="flex-1">
          role, company link, location + dates
        </div>
        <ChevronDown chevron — rotates 180° when open />
      </div>

      {/* Expandable body */}
      <AnimatePresence initial={false}>
        {isOpen && (
          <motion.div
            initial={{ height: 0, opacity: 0 }}
            animate={{ height: 'auto', opacity: 1 }}
            exit={{ height: 0, opacity: 0 }}
            transition={{ duration: 0.25, ease: 'easeInOut' }}
            className="overflow-hidden"
          >
            <div className="pt-3 pl-20"> {/* indent to align with text, past the 64px logo */}
              description paragraph
              tech stack pills
            </div>
          </motion.div>
        )}
      </AnimatePresence>
    </div>
    ```

    **Chevron animation:** Wrap in `<motion.div animate={{ rotate: isOpen ? 180 : 0 }} transition={{ duration: 0.2 }}>`. Use `text-text-muted group-hover:text-text-secondary` for color, size 16.

    **Default state:** `isOpen = false` (collapsed). All cards start collapsed.

    **Avoid:** Don't use `overflow-visible` on the motion.div during exit — keep `overflow-hidden` so content clips cleanly.

    The existing flex layout in experience.tsx is replaced — this is a full rewrite of the component's render output. Preserve all the existing data rendering (role, company, location, dates, logo) in the header section.
  </action>
