<objective>
## Goal
Replace the current endless-scroll layout with a full-viewport shell: a persistent left sidebar showing identity info, and a right panel that swaps content based on the active tab. Five tabs: About, Skills, Work, Projects, Contact. Panel content is placeholder — 02-03 and 02-04 fill them in.

## Purpose
The scroll layout is incompatible with the "Liminal Late Night" direction — a tab/panel architecture creates a dramatically different, more curated feel. Panels keep content focused, navigation is always visible, and the layout sets the stage for Phase 3 animations. This plan establishes the navigation scaffolding that 02-03 and 02-04 build on.

## Output
- `src/components/tab-nav.tsx` — TabNav component (Tab type + vertical tab list)
- `src/app/page.tsx` — Rewritten: full-viewport sidebar + panel shell, tab state, placeholder panels
- Old scroll layout removed; existing section components left untouched
</objective>
