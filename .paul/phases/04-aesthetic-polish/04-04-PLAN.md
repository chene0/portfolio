<task type="auto">
  <name>Task 1: Create StreetlampsScene SVG component</name>
  <files>src/components/streetlamps.tsx</files>
  <action>
    Create a React component (no 'use client' needed — purely static render) that renders an SVG streetlamp scene.

    **File structure:**
    ```tsx
    import React from 'react'
    export function StreetlampsScene() { ... }
    ```

    **SVG setup:**
    - `viewBox="0 0 320 600"`
    - `width="100%" height="100%"`
    - `preserveAspectRatio="xMidYMax meet"`
    - No background fill (transparent, inherits dark page bg)

    **4 lamps — define as an array of objects with these properties:**

    ```
    Lamp 0 (foreground):  cx=145, poleBaseY=600, poleTopY=390, crossbarHalf=65, headW=40, headH=13, glowR=100, opacity=1.0
    Lamp 1 (mid-near):    cx=210, poleBaseY=565, poleTopY=420, crossbarHalf=40, headW=25, headH=8,  glowR=62,  opacity=0.78
    Lamp 2 (mid-far):     cx=248, poleBaseY=530, poleTopY=437, crossbarHalf=25, headW=16, headH=5,  glowR=38,  opacity=0.55
    Lamp 3 (background):  cx=268, poleBaseY=504, poleTopY=449, crossbarHalf=16, headW=10, headH=3,  glowR=24,  opacity=0.36
    ```

    **Per lamp — rendered as a `<g opacity={lamp.opacity}>` group:**

    1. **Pole**: `<rect>` from (cx - poleW/2, poleTopY) to (cx + poleW/2, poleBaseY).
       poleW scales with lamp: lamp0=7, lamp1=5, lamp2=3, lamp3=2. Color: #2a2a2a.

    2. **Crossbar**: `<line>` at y=poleTopY from x=(cx - crossbarHalf) to x=(cx + crossbarHalf). Stroke #2a2a2a, strokeWidth same as poleW.

    3. **Left lamp head**: `<rect>` centered at (cx - crossbarHalf + headW/2 - 2, poleTopY - headH/2 - 8).
       Width=headW, height=headH, rx=2. Fill #f0efe8.

    4. **Right lamp head**: mirror of left, centered at (cx + crossbarHalf - headW/2 + 2, poleTopY - headH/2 - 8).

    5. **Left head glow halo**: `<ellipse>` centered at left head center. rx=glowR*0.7, ry=glowR*0.5.
       Fill: `url(#glow-l{i})`. Use a `<radialGradient>` defined in `<defs>`.

    6. **Right head glow halo**: same as left, mirrored.

    7. **Atmospheric cone**: Wide soft ellipse below the lamp heads. cx=lamp.cx, cy=poleTopY + 80*scale.
       rx=glowR*1.8, ry=glowR*2.2. Fill: `url(#atm-l{i})`.
       This simulates the foggy scatter of light falling downward.

    8. **Ground haze** (lamp 0 only): Very wide ellipse at cy=580, rx=200, ry=40.
       Fill: `url(#ground-haze)`. Simulates light pooling at ground level.

    **SVG `<defs>` — define these gradients:**

    For each lamp `i`, add:
    - `<radialGradient id="glow-l{i}">`: stops: center rgba(225,228,240,0.9) → transparent. `gradientUnits="userSpaceOnUse"` with cx/cy at the respective head center.
    - `<radialGradient id="atm-l{i}">`: stops: rgba(180,195,220,0.12) at 0% → transparent at 100%. Wide ellipse, `gradientUnits="userSpaceOnUse"`.

    For the ground haze:
    - `<radialGradient id="ground-haze">`: rgba(180,195,220,0.08) → transparent. Wide, flat.

    **Rendering order (back to front within scene):**
    Render lamps from index 3 down to 0 so foreground lamp renders on top.

    Avoid: using filter blur (use gradient opacity instead — keeps it GPU-friendly and consistent with CRT overlay).
    Avoid: any animation or event handlers (those come in a future plan).
    Avoid: hardcoding sizes in ways that break at different container aspect ratios.
  </action>
  <verify>
