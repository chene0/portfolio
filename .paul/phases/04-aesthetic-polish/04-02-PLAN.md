## Goal

<task type="auto">
  <name>Task 2: Add all CRT CSS to globals.css</name>
  <files>src/app/globals.css</files>
  <action>
    Append a `/* === CRT Overlay === */` section to the end of `globals.css`.

    **Shared overlay wrapper:**
    ```css
    .crt-overlay {
      position: fixed;
      inset: 0;
      pointer-events: none;
      z-index: 9999;
      overflow: hidden;
    }
    /* All children share these base styles */
    .crt-overlay > div {
      position: absolute;
      inset: 0;
    }
    ```

    **Scanlines** (z-index 10, very subtle):
    ```css
    .crt-scanlines {
      background: repeating-linear-gradient(
        to bottom,
        transparent 0px, transparent 2px,
        rgba(0, 0, 0, 0.07) 2px, rgba(0, 0, 0, 0.07) 4px
      );
      z-index: 10;
    }
    ```

    **Vignette** (z-index 9):
    ```css
    .crt-vignette {
      background: radial-gradient(ellipse at center, transparent 60%, rgba(0, 0, 0, 0.45) 100%);
      z-index: 9;
    }
    ```

    **Moving scan line** (z-index 11, 3px tall band sweeping down):
    ```css
    @keyframes crt-scan-move {
      from { transform: translateY(-100vh); }
      to   { transform: translateY(100vh); }
    }
    .crt-scanline-moving {
      height: 3px;
      background: rgba(255, 255, 255, 0.04);
      z-index: 11;
      animation: crt-scan-move 10s linear infinite;
      will-change: transform;
    }
    ```

    **Film grain** (z-index 12, feTurbulence jitter):
    ```css
    @keyframes crt-grain-jitter {
      0%   { transform: translate(0,     0);    }
      10%  { transform: translate(-2%,  -3%);   }
      20%  { transform: translate(3%,    1%);   }
      30%  { transform: translate(-1%,   2%);   }
      40%  { transform: translate(2%,   -1%);   }
      50%  { transform: translate(-3%,   2%);   }
      60%  { transform: translate(1%,   -2%);   }
      70%  { transform: translate(-2%,   3%);   }
      80%  { transform: translate(3%,   -1%);   }
      90%  { transform: translate(-1%,   1%);   }
      100% { transform: translate(0,     0);    }
    }
    .crt-grain {
      filter: url(#crt-grain);
      opacity: 0.08;
      mix-blend-mode: overlay;
      z-index: 12;
      animation: crt-grain-jitter 0.15s steps(1) infinite;
      will-change: transform;
    }
    ```

    **Flicker** (z-index 13, opacity-only, subliminal):
    ```css
    @keyframes crt-flicker {
      0%, 8%, 92%, 100% { opacity: 0; }
      4%  { opacity: 0.03; }
      96% { opacity: 0.02; }
    }
    .crt-flicker {
      background: rgba(255, 255, 255, 1);
      z-index: 13;
      animation: crt-flicker 8s steps(1) infinite;
    }
    ```

    **Reduced motion override** (disable all animations, keep static effects):
    ```css
    @media (prefers-reduced-motion: reduce) {
      .crt-scanline-moving,
      .crt-grain,
      .crt-flicker {
        animation: none;
      }
    }
    ```

    Avoid: modifying any existing CSS rules. Only append to the end of the file.
  </action>
  <verify>
    `npm run build` (or dev server) starts without CSS errors.

  </verify>
